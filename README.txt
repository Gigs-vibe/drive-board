TASKA — ДОСКА ЗАДАЧ · сборка .exe для Windows 10/11
=====================================================
Версия: 1.3 (обновлено 29.06.2026)
GitHub: https://github.com/Gigs-vibe/drive-board

В ПАПКЕ ДОЛЖНО ЛЕЖАТЬ 6 ФАЙЛОВ:
  app.py            — программа (окно, трей, напоминания)
  board.html        — интерфейс доски (весь UI здесь)
  icon.ico          — иконка .exe
  icon.png          — иконка трея и уведомлений
  requirements.txt  — список зависимостей Python
  build.bat         — сборка в один клик

Или скачай готовый Taska.exe со страницы релизов:
  https://github.com/Gigs-vibe/drive-board/releases

---------------------------------------------------
ВАРИАНТ 1. БЫСТРО (в один клик)
---------------------------------------------------
1. Установи Python с галочкой "Add Python to PATH"
   (если ещё не установлен): https://www.python.org/downloads/
2. Дважды кликни build.bat
3. Дождись надписи "Готово!"
4. Запускай dist\Taska.exe

Можно сразу проверить без сборки:
   python -m pip install -r requirements.txt
   python app.py

---------------------------------------------------
ВАРИАНТ 2. ВРУЧНУЮ (если bat не сработал)
---------------------------------------------------
Открой "Командную строку" (cmd) в этой папке и выполни:

   python -m pip install -r requirements.txt

   python -m PyInstaller --noconfirm --windowed --onefile --name "Taska" ^
     --icon "icon.ico" --add-data "board.html;." ^
     --add-data "icon.png;." --add-data "icon.ico;." app.py

Готовый файл появится в подпапке  dist\Taska.exe

КАК ПОЛЬЗОВАТЬСЯ
---------------------------------------------------
ДОСКА:
• 5 колонок: Бэклог, В работе, Готово, Ежедневные задачи, След. неделя.
• Карточки перетаскиваются мышкой между колонками.
• Наведение мыши на карточку — кнопки Дублировать / Удалить.
• У задачи можно задать приоритет (Низкий / Средний / Высокий),
  срок (дата + время) и описание.
• «Каждый день» в дедлайне = ежедневная задача с напоминанием в заданное время.
• Кнопка сортировки в шапке колонки — сортирует карточки по приоритету.
• Пресеты (меню ⋯ → Пресеты) — набор типовых задач одним кликом.

ПРОГРЕСС И ЦЕЛИ:
• Слева от кольца — надпись «Прогресс» и счётчик «из N задач».
• Кольцо показывает прогресс за день (выполнено / цель).
• Нажми на кольцо → откроется панель «Цели и достижения».
• Выбери дневную цель: 3, 5 или 10 задач.
• 6 ачивок разблокируются автоматически — за количество выполненных задач
  и за серию активных дней (стрик 🔥).
• Стрик 2+ дня подряд → появляется оранжевый бейдж с огнём рядом с кольцом.

АРХИВ:
• Задачи из колонки «Готово» автоматически уходят в архив на следующий день.
• Архив виден в правой панели (вверху, над ежедневными задачами).
• «Посмотреть все» — открывает полный список архива.
• Из архива можно вернуть задачу в Бэклог.

ПРОФИЛЬ И ФОТО:
• Круглая кнопка в правом углу шапки (с инициалами или фото) → Профиль.
• Здесь: имя, фамилия, загрузка фото профиля.
• Имя и фото синхронизируются с Android-приложением Taska через облако
  (при условии входа в один аккаунт).
• Если не вошёл в аккаунт — кнопка «Войти в аккаунт» открывает форму входа.
• Данные профиля сохраняются между перезапусками приложения.

СИНХРОНИЗАЦИЯ С ТЕЛЕФОНОМ:
• После входа в аккаунт доска и профиль автоматически синхронизируются
  через облако — что изменил на ПК, увидишь на телефоне и наоборот.
• Без входа — работает локально на этом устройстве.
• Профиль (имя/фото), установленный на ПК, попадает в облако и
  подтягивается при следующем входе на любом устройстве.

НАСТРОЙКИ (меню ⋯ → Настройки):
• Тема — светлая / тёмная.
• Цвет акцента — 5 пресетов + произвольный цвет (пипетка).
• Цвет фона — белый, розовый, бежевый, лаванда, мятный, небесный
  и произвольный. Доступен в светлой и тёмной теме.
• Дневная цель — 3, 5 или 10 задач.
• Экспорт, импорт, очистка доски, онбординг.

МОЯ КОМАНДА:
• Кнопка «Моя команда» в шапке — раздел для совместной работы (Pro).

УВЕДОМЛЕНИЯ И ТРЕЙ:
• Крестик окна сворачивает программу в трей (значок у часов).
  Полностью выйти: правый клик по значку → «Выход».
• Уведомления приходят за 30 минут до дедлайна и в момент дедлайна,
  ДАЖЕ ЕСЛИ окно свёрнуто.

МОТИВАЦИЯ:
• В баннере — мотивационная фраза и место для GIF/картинки.
  Меню ⋯ → Картинки, GIF и видео — загрузи свой файл или вставь ссылку.
• Кнопка ↺ в правом углу баннера — сменить фразу.

ЭКСПОРТ:
• Меню ⋯ → Отчёт в Excel — выгрузка всех задач + история перемещений.
• Меню ⋯ → Сохранить в файл — резервная копия доски в JSON.
• Меню ⋯ → Загрузить из файла — восстановить из резервной копии.

ДАННЫЕ И СОВМЕСТИМОСТЬ
---------------------------------------------------
• Формат данных совместим с мобильным приложением Taska (Android).
  Войдите в один аккаунт на ПК и телефоне — доска и профиль будут общими.
• Ключи localStorage (браузерное хранилище pywebview):
    kanban-state-v2    — доска (задачи, архив, пресеты)
    drive-profile      — имя, фамилия, фото
    drive-streak       — серия активных дней
    drive-daily-goal   — дневная цель (3/5/10)
    taska-pro-cache    — кеш Pro-статуса (TTL 1 час)
    taska-accent-color — цвет акцента
    taska-bg-color     — цвет фона
    sb-session         — сессия Supabase

---------------------------------------------------
ЕСЛИ ЧТО-ТО НЕ ТАК
---------------------------------------------------
• Окно белое/пустое: убедись, что в Windows есть "WebView2 Runtime".
  На Win 10/11 он обычно уже стоит; если нет — поставь Evergreen
  WebView2 Runtime с сайта Microsoft (бесплатно).
• Антивирус ругается на .exe — это ложное срабатывание PyInstaller.
  Добавь файл в исключения антивируса.
• Уведомления не приходят: Параметры Windows → Система → Уведомления
  — они должны быть включены для «Taska».
• Google-вход не работает: убедись, что localhost:8765 не занят другой
  программой.
• Профиль не синхронизируется: войди в аккаунт, сохрани имя/фото на ПК,
  перезапусти приложение на другом устройстве.
• Данные пропали после обновления: localStorage хранится в
  %APPDATA%\Taska — не удаляй эту папку.
